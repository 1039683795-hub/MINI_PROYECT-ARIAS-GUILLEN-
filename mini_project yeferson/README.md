# Mini Proyecto - Cajero Electrónico

Mini proyecto: Cajero Electrónico en Python

## Descripción

Aplicación de cajero electrónico que permite consultar saldo, depositar y retirar dinero.

## Instalación

```bash
python -m venv .venv
.\.venv\Scripts\activate.bat
python -m pip install --upgrade pip
python -m pip install -e .
```

## Ejecutar

```bash
python programa.py
```

## Pruebas

```bash
python -m pip install pytest
pytest -q
```
